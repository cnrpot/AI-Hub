# AI 中转站聚合平台 — PRD

## 产品简介

一个开源的 AI API 中转站信息聚合与对比平台，帮助开发者和 AI 应用使用者快速发现、比较和选择合适的 AI API 中转服务。

## 问题陈述

当前 AI API 中转站（API 代理/转发服务）数量众多，价格、模型支持、稳定性参差不齐。用户需要在多个站点之间手动比较，信息分散且不透明，缺乏统一的聚合和对比工具。

## 目标用户

- **开发者**：需要接入 AI API 的后端/全栈开发者，关注价格、稳定性和模型覆盖
- **AI 应用构建者**：使用 Dify、LobeChat 等工具的玩家，需要找到性价比高的 API 来源
- **团队决策者**：为公司选择 API 服务的管理者，关注可靠性和成本

## 核心价值主张

一站式的 AI 中转站信息聚合与透明对比，社区驱动维护，完全免费开源。

## 平台目标

- 纯前端静态站点，部署在自有服务器
- 完全免费开源，社区 PR 驱动数据更新
- SEO 友好，方便搜索引擎收录

## 功能列表

### 首页 — 中转站列表

**用户故事：** 作为用户，我想在首页浏览所有收录的 AI 中转站，快速了解每个站的基本信息。

- 展示所有中转站卡片，每张卡片包含：站名、简介、支持模型数、价格区间、评分、特性标签
- 响应式网格布局：移动端 1 列，平板 2 列，桌面 3 列
- 卡片点击进入详情页

### 搜索与筛选

**用户故事：** 作为用户，我想通过关键词搜索、按模型筛选、按价格排序，快速找到符合需求的中转站。

- 搜索框：按站名、描述模糊搜索
- 模型筛选：按 GPT / Claude / Gemini / 其他 多选筛选
- 排序：价格升序/降序、评分降序、更新时间降序
- URL 参数同步：搜索词、筛选条件写入 URL，支持分享链接

### 中转站详情页

**用户故事：** 作为用户，我想查看某个中转站的完整信息，包括支持的模型和详细定价。

- 站名、URL、描述、评分、更新时间
- 定价表格：每个模型的输入价格、输出价格、计费单位
- 支持模型列表
- 特性标签展示
- 外部链接：官网、接入文档

### 多站对比页

**用户故事：** 作为用户，我想同时对比多个中转站的价格和模型覆盖，做出最优选择。

- 在首页卡片上勾选多个中转站
- 对比页展示并排表格：各维度（站名、URL、评分、模型数、各模型价格）
- 模型覆盖矩阵：模型为行、中转站为列，标记支持/不支持
- 同模型最低价格高亮

## 页面清单

1. **首页 /** — Hero + 搜索筛选 + 中转站卡片列表
2. **详情页 /stations/[slug]** — 单个中转站完整信息
3. **对比页 /compare** — 多站并排对比表格
4. **关于页 /about** — 项目介绍 + 贡献指南

## UX 考量

- 深色模式为主，降低长时间浏览的视觉疲劳
- 信息密度适中：卡片展示核心信息，详情页展开完整数据
- 价格数据用等宽字体，便于横向对比
- 对比表格首列 sticky，方便横向滚动时参照维度
- 空状态友好提示
- 加载状态骨架屏
